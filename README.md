# node-boiler-plate
 
 A boiler plate for node express application using typescript, MVC design

 ### tools out of the box
 - Express
 - Typescript
 - Node-pg-migrate
 - postgres, pg,  ORM
 - cookies enabled
 - session eneabled - using express-session
 - Security - {jwt, bcrypt} (you can implement `cors`)

 ### Pull code
 run:
 ```
    yarn 

    yarn start

 ```
